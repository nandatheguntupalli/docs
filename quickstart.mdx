## 1) Install the SDK

Use `uv` (recommended):

```bash
uv add cua-sdk
```

If you want the example backend too, clone this repo and run the backend as below.

## 2) Run your first task

```python
import asyncio
import opcua


async def main():
    result = await opcua.run(
        "Open google.com and search for Operative SDK",
        "claude",
        cua_api_key="YOUR_KEY",
    )
    print(result.success, result.summary)


asyncio.run(main())
```

- **model**: `"claude" | "openai" | "browseruse"`
- Optional: `starting_url="https://example.com"`

## 3) Start the backend (optional but recommended)

The backend proxies model calls and enforces per-key quotas.

```bash
cd CUA-backend
uv sync && uv run python main.py
```

Set `ANTHROPIC_API_KEY` or `OPENAI_API_KEY` in your environment.

## What you get back

`ComputerResult` includes:

```python
result.success        # bool
result.summary        # str
result.screenshots    # List[str] (base64 PNGs)
result.final_message  # str
result.error          # Optional[str]
```

<Tip>Screenshots are base64 PNGs. Store them or stream them to your UI.</Tip>
