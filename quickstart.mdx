## 1) Install the SDK

Use `uv` (recommended):

```bash
uv add cua-sdk
```
 

## 2) Run your first task

```python
import asyncio
import opcua


async def main():
    result = await opcua.run(
        "Open google.com and search for Operative SDK",
        "claude",
        cua_api_key="YOUR_KEY",
    )
    print(result.success, result.summary)


asyncio.run(main())
```

- **model**: `"claude" | "openai" | "browseruse"`
- Optional: `starting_url="https://example.com"`

<Tip>Pass your Operative API key via the `cua_api_key` argument. Keys are required for all providers.</Tip>

### BrowserUse model (optional)

When using `"browseruse"`, you can specify the underlying browser model via `browser_model`.

```python
await opcua.run(
    "Research new features",
    "browseruse",
    cua_api_key="YOUR_KEY",
    browser_model="o4-mini",
)
```

Default is Claude Sonnet 4 (`claude-sonnet-4-20250514`).

## 3) What you get back

`ComputerResult` includes:

```python
result.success        # bool
result.summary        # str
result.screenshots    # List[str] (base64 PNGs)
result.final_message  # str
result.error          # Optional[str]
```

<Tip>Screenshots are base64 PNGs. Store them or stream them to your UI.</Tip>
