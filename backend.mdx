## What the backend does

- Terminates API keys and rate limits with `x-cua-api-key`.
- Proxies chat to providers via LiteLLM.
- Orchestrates remote, step-by-step computer-use sessions.

## Run locally

```bash
cd CUA-backend
uv sync && uv run python main.py
```

Environment:

- `ANTHROPIC_API_KEY` for Claude
- `OPENAI_API_KEY` for OpenAI (optional)
- Optional Supabase for persistent quotas

## Endpoints

- `POST /remote/start` → returns first action
- `POST /remote/step` → returns next action or completion
- `POST /proxy/litellm/chat` → OpenAI-style chat via LiteLLM

Use with the SDK. The browser always runs locally; the server never controls your browser.
